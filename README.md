# 🎯 Pattern Generator and Number Analyzer

A simple Python console application that allows users to:

- Generate a star (`*`) pattern with a specified number of rows
- Analyze a range of numbers and determine whether each number is even or odd
- Calculate the sum of all numbers within a given range
- Interact through an easy-to-use menu-driven interface

---

## 📌 Features

### ⭐ Pattern Generator
- Creates a right-angled triangle pattern using asterisks (`*`)
- User chooses the number of rows

### 🔢 Number Analyzer
- Checks every number in a given range
- Identifies whether each number is **Even** or **Odd**
- Calculates the total sum of all numbers in the range

### 🚪 Exit Option
- Safely exits the program when selected
